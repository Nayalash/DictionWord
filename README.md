## Commands

* GET information regarding the discord bot.

```
!help
```

* GET the definition of a word

```
!fetch (word)

Example:

!fetch family
```

Example Use

```
> !fetch family

OUTPUT:

The definition for that word is: A group consisting of two parents
and their children living together as a unit.

```
